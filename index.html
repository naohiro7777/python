Pandas

Pandasは、主に、データの読み込みができます。

Pandasのインストール方法は、Windowsであれば、コマンドプロンプト、MacOSであれば、ターミナルを開き以下のコードを打ち込んでください。

pip install pandas

打ち込んだコマンド

それができたらWindowsであればenterキーMacOSであれば、returnキーを押してくださいそうすればインストール完了です。
